# Claims

1. Tactile residuals can be used as a causal diagnostic channel, not just a reactive alarm.
2. Under a contact-structured predictor, residual shape and timing can separate expected model error from unexpected external contact better than raw thresholding.
3. This distinction matters because the two cases call for different robot responses: model adaptation vs interruption / recovery.
4. Existing tactile/contact papers mostly estimate state, detect slip, or improve control; they do not center the residual as an explanatory variable with a causal role.
5. The paper is strongest if it demonstrates the failure of a pure prediction-error baseline under exogenous contact and shows an operational separation signal.
